# Data Integra Finance

Projeto React desenvolvido na empresa DataIntegra , com o objetivo de ensinar e compartilhar conhecimento.

## O que é o DataIntegra Finance?

É um projeto que faz o controle sobre receitas e despesas, onde o usuário pode cadastrar suas receitas e despesas, e acompanhar o saldo de sua conta.

## Tecnologias

- [React](https://pt-br.reactjs.org/) - Biblioteca JavaScript para criar interfaces de usuário
- [React Router](https://reactrouter.com/) - Biblioteca para gerenciar rotas
- [Emotion](https://emotion.sh/) - Biblioteca para estilização de componentes
- [React Hook Form](https://react-hook-form.com/) - Biblioteca para gerenciar formulários
- [Material UI](https://material-ui.com/) - Biblioteca de componentes React para desenvolvimento web
- [Vite](https://vitejs.dev/) - Ferramenta de build para desenvolvimento web
- [Axios](https://axios-http.com/docs/intro) - Biblioteca para fazer requisições HTTP para API

## Como executar o projeto

### Pré-requisitos

- [Node.js](https://nodejs.org/en/) - Versão LTS

### Instalação

```bash
# Clone este repositório
$ git clone

# Acesse a pasta do projeto no terminal/cmd
$ cd web_dIntegra

# Instale as dependências
$ npm install

# Execute a aplicação em modo de desenvolvimento
$ npm run dev
```
